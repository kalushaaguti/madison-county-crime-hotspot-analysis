# Crime Hotspot Analysis in Madison County, Illinois (2020–2024)

## Overview
This project analyzes spatial patterns of violent crime in Madison County, Illinois, focusing on **Robbery** and **Aggravated Assault** from 2020 to 2024 using ArcGIS Pro.

The goal is to identify statistically significant hotspots and understand how crime patterns change over time.

---

## Study Area
Madison County, Illinois, USA

---

## Data Sources
- Illinois State Police (Crime Data, 2020–2024)
- U.S. Census Bureau (Boundary Shapefiles)

---

## Tools Used
- ArcGIS Pro
- Spatial Statistics Tools (Hotspot Analysis)
- Geospatial Data Processing

---

## Methodology

### Data Preparation
- Organized crime data by year (2020–2024)
- Selected:
  - Robbery
  - Aggravated Assault
- Joined crime tables to county boundary shapefile
- Replaced missing values with 0

### Hotspot Analysis
- Tool: **Optimized Hot Spot Analysis**
- Distance Band: **1 mile**
- Generated hotspots and cold spots for each year

### Visualization
- Color ramp: **Red → Yellow → Green**
- Consistent classification across all years

## Key Findings

### Robbery
- Persistent hotspots: **Alton, Granite City**
- Emerging area: **Wood River (2024)**
- Cold spots: **Livingston (consistent)**

### Aggravated Assault
- Strong hotspots: **Alton, Granite City**
- Increasing trends: **Wood River, Edwardsville**
- Stable cold spot: **Livingston**
